icestorm-bin Documentation¶

icestorm-bin is a portable, pre-built distribution of Project IceStorm — the open-source toolchain for Lattice iCE40 FPGAs — for Linux (manylinux 2.34 / glibc 2.34+). It provides ready-to-use binaries for packing, unpacking, and analysing iCE40 bitstreams in a single self-contained tarball.
